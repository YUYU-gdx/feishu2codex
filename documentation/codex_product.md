# Codex product overview

Source: https://openai.com/codex/

Summary:
High-level overview of Codex as OpenAI's AI coding partner, with emphasis on agentic workflows, multi-agent parallelism, and integrations across app, IDE, and CLI.

Key points:
- Codex runs tasks end-to-end, including refactors, migrations, and PR workflows.
- Designed for multi-agent workflows and background automations.
- Available across app, IDE, and CLI (npm package @openai/codex).
